# goldpecha(골드폐차)

> 폐차에 관한 문의 상담 페이지

## Spec

> VueJS, scss, webpack, express, mongoose, nodeJS

## build setUp

``` bash
# install dependencies
npm install or yarn 

# serve with hot reload at localhost:8080
npm run dev or yarn dev

# build for production with minification
npm run build or yarn build

```
